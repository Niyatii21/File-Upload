# File Upload to Dropbox

A Node.js project to upload files to Dropbox using the Dropbox API.

## Features
- Select and upload files through a web interface.
- Store files in Dropbox using the Dropbox API.

## Technologies Used
- Node.js
- Express
- Dropbox API
